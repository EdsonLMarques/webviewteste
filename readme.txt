anotações sobre pywebview

exutando ações a partir do front utilizar o seguinte passo a passo
    criar classe
    na classe, criar os esqueletos das funções (chamada de codigo + retorno)

    no html, criar uma função js
        dentro da função, executar o comando pywebview.api.NOME_DA_FUNÇAO isso ira executar a funçao esqueleto em python



a função evaluate_js serve para executar um codigo JS a partir do python, o codigo deve estar no formato string

a função expose serve para criar mais uma conexao de API dessa forma é possivel gearar
